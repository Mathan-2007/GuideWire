# GIG-SHIELD 🚀

AI-Powered Parametric Insurance Platform for Gig Workers

---

## 📌 Overview

GIG-SHIELD is a smart insurance system designed for gig economy workers such as delivery riders (Swiggy, Zomato, Blinkit).

The platform provides **instant, automated payouts** based on real-world conditions like rainfall, AQI, and wind speed — eliminating the need for manual claim filing.

---

## 👤 Personas & Use Cases

### 1. Delivery Rider (Primary User)

* Works in dynamic environmental conditions
* Faces income risk due to rain, pollution, or disruptions
* Needs instant compensation without paperwork

**Scenario:**

> Arjun, a Swiggy rider, is working in heavy rainfall (>20mm/hr).
> GIG-SHIELD automatically detects this and credits ₹500 to his wallet.

---

### 2. Admin (Platform Operator)

* Monitors zones and risks
* Tracks payouts and fraud
* Adjusts trigger thresholds

**Scenario:**

> Admin sees multiple fraud alerts from one rider and flags the account instantly.

---

## ⚙️ Workflow

1. Rider enrolls and pays weekly premium
2. System continuously monitors:

   * Weather APIs
   * AQI data
   * GPS telemetry
3. When threshold is breached:

   * Trigger fires
   * Eligible riders identified
   * Instant payout processed
4. AI system checks for fraud anomalies
5. Admin dashboard displays everything in real-time

---

## 💰 Weekly Premium Model

* Riders pay a **fixed weekly premium (~₹47)**
* Premium calculated based on:

  * Location risk
  * Historical weather data
  * Rider activity

### Why Weekly?

* Matches gig worker income cycle
* Affordable and flexible
* Encourages continuous coverage

---

## ⚡ Parametric Triggers

| Trigger Type  | Condition  | Payout |
| ------------- | ---------- | ------ |
| Heavy Rain    | ≥ 20 mm/hr | ₹500   |
| Moderate Rain | ≥ 12 mm/hr | ₹150   |
| Severe AQI    | ≥ 300      | ₹500   |
| High Wind     | ≥ 40 km/h  | ₹150   |

👉 These triggers ensure:

* No claim filing
* No delays
* Fully automated payouts
